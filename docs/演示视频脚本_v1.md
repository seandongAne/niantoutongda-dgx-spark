## 一句话故事

搬家最痛的不是搬,是搬完找不到东西——对着旧屋随手拍的几段视频和一段旁白,
Agent 记住每件物品、听懂去向、理解新家相对空间、规划摆位并开出可审计任务卡；
如需证明真实摆放执行,再追加一张搬后照片复核。
