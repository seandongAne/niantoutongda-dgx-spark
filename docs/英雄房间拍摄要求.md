# 英雄房间拍摄要求(给拍摄队友)

> 这批视频是初赛演示的**唯一素材来源**:AI 要从你拍的旧房间视频里认出每件物品、
> 记住它们的分组,再规划搬到"新家"后的摆放。拍摄质量直接决定演示效果。
> 全部素材只在我们自己的机器上处理,不上传任何云端。

## 一、房间怎么布置(拍之前先整理,10 分钟)

- **整洁干净**:桌面、地面、架子先收拾一遍,和演示无关的杂物全部清出画面。
- **物品 12~15 件**,每件轮廓清楚、尺寸适中(矿泉水瓶到台灯这个量级最好,
  别用太小的橡皮/数据线当主角)。
- **绝对不要叠放、堆放、互相遮挡**——每件物品之间留出一掌以上的空隙,
  一眼能看出"这是独立的一件东西"。
- **刻意放 1~2 组"同款不同色"**:比如一个粉色水壶 + 一个蓝色水壶,
  或两个颜色不同的收纳盒。这是演示的加分项(AI 要证明能分清同类的两件)。
- **摆出三个"生活组合"**,每组 3~4 件,组与组之间空间上明显分开:
  - 睡前组合(例:台灯、水壶、毛绒玩具、一本书 —— 床头区域)
  - 学习组合(例:笔筒、台灯、书、平板支架 —— 书桌区域)
  - 收纳/充电组合(例:收纳盒、充电器、耳机、数据线篮 —— 柜子/角落区域)
- 光线:**白天、开灯、光线稳定**。避免逆光(别让镜头正对窗户拍)。

## 二、拍什么(共 4 段视频 + 1 段旁白)

| # | 内容 | 时长 | 要点 |
|---|---|---|---|
| 1 | 旧房间引导视频 第 1 段 | 1.5~3 分钟 | 从门口开始,按下面"怎么拍"走完全程 |
| 2 | 旧房间引导视频 第 2 段 | 1.5~3 分钟 | **换一个起点和行走方向**再拍一遍(比如从窗边开始、逆时针走) |
| 3 | 旧房间引导视频 第 3 段 | 1.5~3 分钟 | 再换一个起点/路线;三段的拍摄顺序、角度要有差异,但物品位置不动 |
| 4 | "新家"房间视频 | 1~2 分钟 | 另一个房间(或同房间清空重布置的一角)当"新家":扫一遍将来可以摆东西的区域——书桌面、床头柜、架子、空柜格,每个区域停留 2 秒 |
| 5 | 口述旁白 | 2~3 分钟 | 手机录音即可(m4a/wav)。对着物品清单逐件说:**这是什么、是谁的、现在在哪、搬到新家想放哪、和什么打包在一起**。例:"蓝色水壶,妹妹的,现在在书桌上,搬过去放床头,和台灯一组。" |

每件物品必须**至少在三段旧房间视频中的两段里完整出现过**——最稳的做法是
每段都把 12~15 件全拍到。

## 三、怎么拍(手机即可,横屏)

1. **横屏**持机,双手端稳;默认相机就行,1080p 或 4K 均可,**关掉美颜/滤镜/夜景模式**。
2. **慢**——移动速度比你觉得的慢一倍,转身时尤其慢(快速甩动会糊)。
3. 每段先**远景绕一圈**:让整个房间和三个组合区域都入画;
   再**近景走一圈**:走到每件物品面前,**让它完整进入画面,停住 1~2 秒**,再走向下一件。
4. 物品要**完整入画**(别只拍到半个),尽量在画面中央,距离 0.5~1.5 米。
5. 一镜到底,中途不暂停、不切换摄像头、不变焦(用走近代替 zoom)。
6. 画面里**不要出现人脸、证件、快递单**这类隐私内容。

## 四、拍完怎么交

- 文件命名:`old_1.mp4 / old_2.mp4 / old_3.mp4 / new_1.mp4 / narration.m4a`。
- **传原文件**:AirDrop / 网盘 / 数据线;**不要走微信**(会压缩画质,等于白拍)。
- 顺手发两张房间的普通照片(全景 + 任意角度),方便我们核对物品清单。

## 五、发送前自查清单

- [ ] 12~15 件物品,无叠放、无遮挡、间距一掌以上
- [ ] 有 1~2 组"同款不同色"
- [ ] 三个生活组合肉眼可辨、空间分开
- [ ] 旧房间 3 段,起点/路线各不相同,物品位置三段之间没动过
- [ ] 每件物品在至少两段里完整出现并停留过 1~2 秒
- [ ] 新家 1 段,候选摆放区域都扫到
- [ ] 旁白逐件说了"是什么/谁的/在哪/去哪/和谁一组"
- [ ] 横屏、白天、无逆光、无滤镜、画面里无隐私信息
- [ ] 原文件传输,不经微信

拍摄中拿不准的,宁可多拍一段备用——重拍比补拍容易,素材多不嫌多。
