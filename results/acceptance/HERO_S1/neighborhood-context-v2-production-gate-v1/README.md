## 结论

单锚点 v2 在现有无 GT 生产门上通过，但保持实验状态，不替换当前主工作点。

v2 不再把“一个邻居”当作强确认：目标-锚点距离同时按画面对角线和同帧锚点间
尺度归一化，跨帧 MAD、共视率与共同帧数形成置信度；一个共同锚点的基础置信度
上限为三分之一，实际 blend 再乘该置信度。

## 单锚点是否有独立贡献

有运行指标贡献，但不是“凭一个邻居直接确认实体”。同算法、同 blend 的三锚点
对照为链接 994/澄清 589；允许 1–2 锚点后为 996/589，净增加 2 条 cycle closure。
其中一条闭环直接使用 1 锚点和 2 锚点两条 support pair；另一条来自扩大的校准
候选集合对 Hungarian 排序的间接影响。新增的 2 条 automatic 自身仍有 5–6 个
共同锚点。

## 精度边界

配置冻结后，对已经使用过的 20-anchor GT 只做事后安全回归：完整合并保持
`17/20`，高置信跨锚误合并保持 0，R@1 从 `488/577=0.8458` 到
`489/577=0.8475`，增加 1 个 `anchor_05` 命中，但仍低于 0.86 门。该 GT 不是
独立测试集，且没有硬负组，因此只能说明已知样本未退化，不能证明泛化精度提升。
