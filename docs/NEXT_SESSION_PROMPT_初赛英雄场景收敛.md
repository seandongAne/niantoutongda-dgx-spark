## 初赛唯一目标

设计并跑通**一个授权、整洁、受控的卧室搬家英雄任务**：

```text
三段旧家引导视频
→ 关键帧 / GDINO 候选 / 单视频轨迹
→ 跨视频实体聚合
→ 睡前组合 / 学习组合 / 收纳或充电组合
→ 新家候选区域
→ CP-SAT 区域级布局
→ 结构化任务卡
→ 完成图与双证据验收 trace
```

演示必须是真实管线成果，但不承担跨家庭、跨房型、任意物品泛化证明。

## 英雄场景设计合同

- 只保留一个初赛英雄任务；现有 dev_a 可继续作为压力测试和技术演进素材，不扩成多房间矩阵。
- 12～15 件锚点，轮廓清楚、尺寸适中、互不遮挡、不堆叠。
- 每件锚点至少出现在三段视频中的两段；完整入画，停留约 1～2 秒。
- 光线稳定、慢速平移，先拍房间关系远景，再拍物品清晰近景。
- 三个生活组合各 3～4 件物品，组合边界清楚、彼此留空：
  - 睡前组合示例：床头灯、水杯、纸巾或书；
  - 学习组合示例：台灯、笔筒、书、本子或充电器；
  - 收纳/充电组合示例：收纳盒、设备、线材、电源附近区域。

- 新家区域保持清楚：床头表面、书桌近插座区、柜体或收纳区等候选位置应能被稳定看见。
